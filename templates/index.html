Instructions

How to use this email automation tool

Excel File Format

Your Excel/CSV file must contain these columns:

  • To - Recipient email (Required)
  • Subject - Email subject (Required)
  • Body - Email message body (Required)
  • CC - Carbon copy recipients (Optional)
  • BCC - Blind carbon copy (Optional)
  • Attachment - File path to attachment (Optional)

Example Data

To Subject Body CC
john@example.com Hello John This is a test email jane@example.com
jane@example.com Hello Jane Another test message

Security Tips

  • Always use App Passwords, never your actual Gmail password
  • Enable 2-Factor Authentication on your Google account
  • Keep your credentials confidential
  • Test with a small batch before sending to large lists
  • Respect email sending limits (Gmail: ~500/day)

Getting Started

  1. Prepare your Excel file with required columns
  2. Upload the file or connect Google Sheets
  3. Configure your email settings
  4. Test with a small sample first
  5. Start sending your emails
  6. Monitor logs for status updates
